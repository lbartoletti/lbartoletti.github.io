# QGIS dependencies graph

Here are three graphs representing the dependencies needed to compile QGIS (on FreeBSD 12 using the default compile options). Note that QGIS server is not compiled by default on FreeBSD, you should add the www/fcgi dependency.

The first represents the graph of the dependencies necessary for compilation, the second for execution and the third mixing the two.

- [QGIS run](./qgis_run.svg)
